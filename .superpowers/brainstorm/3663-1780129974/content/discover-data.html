发现首页的 Featured 卡片，数据从哪来?

Codex 的 Featured 是官方后台精选。codeshell 没有这个服务，所以「精选什么」得有个来源。三个可行方案：

A

从已添加的市场聚合

Featured = 已添加市场里的插件（取前 N 个或带 category 的）。没加市场时显示空态引导「去添加市场」。真实、无需额外数据源。推荐

B

内置一份精选清单（本地 JSON）

仓库里放一个 featured.json，列几个推荐的 GitHub skill/插件源。可控但要人工维护，且数据会过时

C

首页不做 Featured 网格

首页只保留大标题 + banner + 搜索 + 「我已安装的」概览，发现/安装全交给市场 tab。最省，但离 Codex 首页最远

另外 banner（紫色轮播区）你想要它干嘛？点完上面顺带在终端说一句：要 banner 放推荐/上手引导，还是先不做 banner。
